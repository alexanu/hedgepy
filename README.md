hedgepy
===============================

version number: 0.0.1
author: Maya Benowitz

Overview
--------

Nowcasting non-linear market correlations for portfolio construction and management.

Installation / Usage
--------------------

To install use pip:

    $ pip install hedgepy


Or clone the repo:

    $ git clone https://github.com/mayabenowitz/hedgepy.git
    $ python setup.py install
